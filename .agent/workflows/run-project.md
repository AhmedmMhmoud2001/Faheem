---
description: كيفية تشغيل مشروع الفهيم
---

1. تأكد من وجودك في مجلد المشروع في التيرمينال.
2. قم بتشغيل الأمر التالي:
```bash
npm run dev
```
3. انتظر حتى يظهر لك رابط مثل `http://localhost:5173/` (أو منفذ آخر إذا كان 5173 مشغولاً).
4. اضغط على الرابط لفتح المشروع في المتصفح.

**ملاحظة:** إذا واجهت مشكلة "Failed to reload"، قم بتحديث الصفحة يدوياً (Refresh) في المتصفح.
